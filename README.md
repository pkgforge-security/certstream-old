- #### About:
> - A very simple CLI Implementation of [CaliDog/certstream-go](https://github.com/CaliDog/certstream-go) Library with [some QOL changes](https://github.com/pkgforge-security/certstream-old/blob/main/main.go)
```bash
!# Install CLI
!# With root
sudo curl -qfsSL "https://bin.pkgforge.dev/$(uname -m)-$(uname -s)/certstream" -o "/usr/local/bin/certstream" && sudo chmod +xwr "/usr/local/bin/certstream"
sudo eget "https://bin.pkgforge.dev/$(uname -m)-$(uname -s)/certstream" --to "/usr/local/bin/certstream"

!# Without ROOT
curl -qfsSL "https://bin.pkgforge.dev/$(uname -m)-$(uname -s)/certstream" -o "$HOME/bin/certstream" && chmod +xwr "$HOME/bin/certstream"
eget "https://bin.pkgforge.dev/$(uname -m)-$(uname -s)/certstream" --to "$HOME/bin/certstream"

!# Using go
go install -v "github.com/pkgforge-security/certstream-old@main"
```

- #### [CLI (Client)]()
```bash
!# CLI Usage:
certstream `{OPTS}`
  -url             --> Certstream Server WebSocket URL (ws:// | wss://)      # Use: -url "wss://certstream.calidog.io" if you don't have your own server
  -domains-only    --> Extract and Print only Domains from Certstream Output #Recommended if you don't want to overload the cli (Raw JSON is hugeeee)
  -insecure        --> Allows Invalid/Insecure SSL Connections, for wss://
  -quiet           --> Suppress Standard Error Output (Useful for Automation)
  -skip-heartbeats --> Skip Sending Heartbeat (Ping) Messages to Certstream Server #Some servers will disconnect you if you don't ping

!# Example:
certstream -url "wss://certstream.calidog.io" -domains-only -quiet
```

```bash
!# Client
certstream -url "ws://localhost:8888" -domains-only -quiet
```
